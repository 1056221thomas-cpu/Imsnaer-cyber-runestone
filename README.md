Ímsnær lét ræisa stæin þansi fyrir Vitiuar, félaga sinn. Byrr sé hánum at baki, vaxi fé ok orðstírr. Æ mun standa, með stæin lifiʀ, vitring sú. Þórr vígi þassi rúnar.
